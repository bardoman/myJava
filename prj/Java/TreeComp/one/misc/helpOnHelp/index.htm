Using the Index

Click the Index tab in the Navigator window to display the keyword index.

To find a topic using the index:

  1. In the field at the top, type the first few letters of a keyword for which you want to search. The keyword list scroll to the relevant keywords as you type. As keywords are selected, one or more topics will appear in the topics list below.

  2. Display a topic from the topics list:
    • Double-click a topic.
    • Select a topic, then click the Open button.
    • Select a topic, then click .
    • To display a topic in a new window, select a topic, then click .

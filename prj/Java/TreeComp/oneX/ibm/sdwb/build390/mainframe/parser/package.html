parsers for mainframe verbs from the preprocessor.
Parsers to convert the return reports from mainframese to more understandable
Java classes that are easier to access. Right now there are parsers scattered in
several different locations.  They should gradually be moved here as time
permits.  Hopefully, someday (when the sun has exhausted it's fuel and
expanded out to earth's orbit) the mainframe will use a standard encoding
(such as XML) so that only one parser will be needed with different wrappers
to access the encoded information.

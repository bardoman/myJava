data classes that are used to pass information between different packages.
Classes should only go here if they are used to store and transfer data between different packages.
An information class that stored information about processes and nothing else should go in the process
package.  A class that stores information on files which is used in library classes and process classes
should go here.  This is not a horribly well defined package.  In general, if you think something
should go here, try to find a place it fits in better instead.  Only if you can't do that put it
here.

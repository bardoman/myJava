Using Search

Click the Search tab in the Navigator window to display the full-text search.

  1. Enter your search criteria and click the Search button:
    • Type the word or words you want to search for in the field at the top.
    • Check Case-sensitive to match the case of these words.
    • Select one of the following:
      • Any of these words -- displays topics that contain at least one of the search words.
      • All of these words -- dislays only the topics that contain all of the search words.
      • This boolean expression -- uses the search words as a boolean expression. Help recognizes AND, OR, and NOT operators.

  2. Matching topics appear in the topics list sorted by relevance -- indicated by the following icons:
    -- indicates a high level of relevance
    -- indicates a medium level of relevance
    -- indicates a low level of relevance

  3. By clicking on the headers of the topics list, sort the results by relevance, topic title, or source.

  4. Display a topic from the topics list:
    • Double-click a topic.
    • Select a topic, then click the Open button.
    • Select a topic, then click .
    • To display a topic in a new window, select a topic, then click .

Note: You can use previous search criteria by selecting the keywords from the drop-down list where you enter the word or words for the search.
